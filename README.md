# cli-chat

> wechat in terminal

## 📦 Installation

```bash
make release

release/cli-chat
```

## ⚡️ Usage

- 使用`tab` 键切换面板, 面板从左到右依次分别为 最近联系人(暂未实现功能), 对话框, 通讯录

- 通讯录通过上下键切换联系人, `enter` 键选中联系人

- 在对话框使用`ctrl+s`发送消息, 目前对话长度超过对话框面板会清空历史消息, 后续会解决
